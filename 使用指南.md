# 中国古代建筑结构可视化项目 - 使用指南

## 🚀 快速开始

### 方式一：直接打开（最简单）

1. 下载或克隆本项目
2. 双击打开 `index.html` 文件
3. 在浏览器中查看可视化效果

**注意**：需要联网加载 ECharts 库（CDN）

### 方式二：本地服务器（推荐）

**使用 Python（通常已预装）：**
```bash
cd ancient-architecture-visualization
python -m http.server 8000
```
然后访问：http://localhost:8000

**使用 Node.js（需安装 http-server）：**
```bash
cd ancient-architecture-visualization
npx http-server -p 8000 -o
```

## 🎨 修改图表样式

### 修改步骤

1. 打开 `js/charts.js` 文件
2. 找到对应的图表初始化函数
3. 修改 ECharts `option` 配置对象
4. 保存文件
5. 刷新浏览器

### 常用配置项示例

#### 修改图表标题
```javascript
var option = {
    title: {
        text: '你的新标题',  // 修改这里
        left: 'center',
        textStyle: {
            fontSize: 18,    // 字体大小
            color: '#2c3e50' // 颜色
        }
    },
    // ... 其他配置
};
```

#### 修改颜色
```javascript
series: [{
    type: 'radar',
    data: [...],
    itemStyle: {
        color: '#your-color'  // 修改颜色
    }
}],
color: ['#c0392b', '#3498db']  // 修改系列颜色数组
```

#### 修改图表尺寸
```javascript
// 在 HTML 中修改 .chart-box 的高度
.chart-box {
    width: 100%;
    height: 500px;  // 修改这里
}

// 或通过 JavaScript 动态调整
chart.resize();
```

## 🔍 常见问题排查

### Q1: 图表不显示，空白区域
**可能原因：**
- ECharts 库未加载（网络问题）
- JavaScript 语法错误
- 容器元素不存在

**解决方法：**
1. 打开浏览器开发者工具（F12）
2. 查看 Console 是否有错误
3. 检查 Network 标签页，确认 echarts.min.js 加载成功
4. 确认 HTML 中的 `div` ID 与 JS 中的 `getElementById` 一致

### Q2: 图表显示但数据不对
**可能原因：**
- `data.js` 数据格式错误
- 数据数组长度不匹配

**解决方法：**
1. 检查 `data.js` 是否有语法错误（缺少逗号、括号不匹配）
2. 对比 `data.js` 中的数组长度是否一致
3. 在浏览器 Console 输入数据变量名查看内容

### Q3: 样式乱码/错位
**可能原因：**
- CSS 文件未加载
- 浏览器缓存旧版本

**解决方法：**
1. 强制刷新（Ctrl+F5 或 Cmd+Shift+R）
2. 清除浏览器缓存
3. 检查 CSS 文件路径是否正确

### Q4: 导出图表功能无效
**可能原因：**
- 浏览器阻止下载
- 图表尚未初始化完成

**解决方法：**
1. 手动允许下载权限
2. 确保图表已完全加载（看到图表再点导出）
3. 尝试其他浏览器（Chrome 效果最佳）
